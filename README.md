HappinesSQL은 사용자의 일상 활동, 기분, 위치 등을 기반으로 행복을 분석하여 개인화된 행복 리포트를 제공하는 서비스입니다. 사용자가 자신의 일상 활동을 기록하고 분석함으로써 언제, 어디서, 무엇을 하며 가장 큰 행복을 느끼는지 파악할 수 있도록 돕는 것을 목적으로 합니다.
